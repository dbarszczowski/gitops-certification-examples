CONFIGURE APP FOR KUBESEAL:
REPO URL: https://bitnami-labs.github.io/sealed-secrets    
CHART: sealed-secrets
version:  2.13.4


Install kubeseal as admin in Powershell:
https://github.com/jordan-owen/kubeseal-windows-installer/blob/main/Install-Kubeseal.ps1

Setup possibility to use CLI for all Users:
# Adding the installation directory to the PATH environment variable for all users
[Environment]::SetEnvironmentVariable("PATH", "$binDir;$env:PATH", [System.EnvironmentVariableTarget]::Machine)



Encrypt file:
Login to correct cluster where kubeseal is installed:
az account set --subscription="R2S11_KGIT_AS-Platform-Operations_DEV"

az aks get-credentials -g Kubernetes_rg --name <CLUSTER_NAME> --admin --overwrite-existing
kubect get svc -A 		OR 		kubectl get svc -A | Select-String -Pattern "sealed-secrets"


ENCRYPT A FILE:
$dbcontent = Get-Content .\db-creds.yml
$sealedSecret = $cbcontent | kubeseal --controller-name controller-test-sealed-secrets --controller-namespace dawid-app -o yaml
$sealedSecret > db-creds-encrypted.yaml

Get-Content .\db-creds.yml | kubeseal --controller-name controller-test-sealed-secrets --controller-namespace dawid-app --format=yaml | Out-File .\db-creds-encrypted.yaml
Get-Content .\paypal-cert.yml | kubeseal --controller-name controller-test-sealed-secrets --controller-namespace dawid-app --format=yaml | Out-File .\paypal-cert-encrypted.yaml
